# Diabetic Retinopathy Dataset

This dataset contains retinal fundus images for identifying **Diabetic Retinopathy (DR)**.

## Classes
- **DR** – Images showing diabetic retinopathy
- **No DR** – Images of healthy retina

## Purpose
Used for image classification and deep learning projects related to eye disease detection.

## Image Type
- Color retinal fundus images

## Applications
- Diabetic Retinopathy detection
- Medical image analysis
- Computer vision projects
